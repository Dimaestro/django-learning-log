Написание спецификации

    В полной спецификации описываются цели проекта, его функциональность, а также внешний вид и интерфейс пользователя.

Создание виртуальной среды и активация :

    $ python3 -m venv ll_env                             - создание 
    $ source ll_env/bin/activate                         - активация виртуальной среды
    $ deactivate                                         - завершение работы среды

Установка Django :

    $ pip install Django

Создание проекта :

    $ django-admin.py startproject learning_log .        -Точка в конце команды создает новый проект со структурой каталогов, которая      упрощает развертывание приложения на сервере после завершения разработки.   

База данных для проекта :

    $ python manage.py migrate                           - миграции (создание базы данных SQLight)

Запуск сервера :

    $ python manage.py runserver
    $ python manage .py runserver 8001                   - если порт уже используется 

Создание приложения :

    $ python manage.py startapp learning_logs
